# Mall-System

> A Vue.js SPA based on vue-cli
+ Plugins: vue-router + axios + vuex
+ Backend: Node.js + express + MongoDB

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
