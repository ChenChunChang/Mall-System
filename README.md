# Mall-System

> A Vue.js SPA based on vue-cli
+ Plugins: 
  - vue-router + axios + vuex 
  - vue-lazyload + vue-infinite-scroll
+ Backend:  
  - Node.js + Express + MongoDB(mongoose)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

```
